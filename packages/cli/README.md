## 📦 Installation

```bash
curl -fsSL https://github.com/hieudoanm/transcript/raw/refs/heads/master/packages/cli/bin/transcript -o transcript
chmod +x transcript
sudo mv transcript /usr/local/bin/
```

## 🚀 Usage

```bash
transcript
transcript fetch <video-id-or-url>
```
